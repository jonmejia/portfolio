## Getting Started

1. Clone the Repository
```
git clone git@github.com:jonmejia/portfolio.git
cd portfolio/ && npm install
```

3. Start Development Server
```
npm run dev
```

Open [http://localhost:4000](http://localhost:4000) with your browser to see the result.
